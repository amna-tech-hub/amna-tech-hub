# 💫 About Me:
Hi! I'm Amna Abbas, a Computer Science student at KFUEIT (CGPA 3.69) and a passionate Full-Stack AI & MERN Developer. I specialize in building responsive, high-performance web applications using React, Redux Toolkit, Node.js, Express, and MongoDB, while bringing static apps to life by integrating modern AI models like Gemini and Llama.  Currently, I'm working on combining full-stack web architectures with LLMs and vector databases. My hands-on experience ranges from developing an Airbnb Clone deployed on Vercel to collaborating in agile teams on live e-commerce platforms like AHM Fragrances. I'm eager to collaborate on full-stack AI tools, open-source projects, and junior/internship opportunities where I can bridge traditional web development with smart AI integration!
